# MixMaster v4

Enhanced version with advanced features and elegant design patterns.

## New in v4

- **Smoker Lab** - Wood profiles with intensity ratings and guided smoking timers
- **People Profiles** - Track taste preferences for personalized recommendations
- **Enhanced UI** - Card-based layouts with shadcn/ui design patterns
- **Detailed Information** - Technique, difficulty, ABV, and pro tips for each cocktail
- **Polished Design** - Dark theme with elegant information hierarchy

## Tech Stack

- React 19 with Hooks
- shadcn/ui design patterns
- Mobile-first responsive design
- TheCocktailDB API integration

## Architecture Notes

v4 uses a more sophisticated information architecture:
- Card components for organized content presentation
- Detailed data structures for wood profiles and people preferences
- Enhanced state management for smoker timers
- Improved modal designs for better UX
