# MixMaster - Cocktail App

A modern cocktail recipe app with advanced features including ingredient matching, smoker lab, and personalized recommendations.

## Project Versions

### v3 - Initial Feature-Complete Version
Located in `/v3` - This version includes:
- TheCocktailDB API integration (636+ cocktails with images)
- "What Can I Make?" ingredient matching
- Advanced search and filtering
- Favorites system
- My Bar inventory tracking
- Basic UI with bottom navigation

### v4 - Enhanced Design Version
Located in `/v4` - Builds on v3 with:
- Smoker Lab with wood profiles and guided timers
- People profiles for personalized recommendations
- Elegant card-based UI with shadcn/ui design patterns
- Detailed cocktail information (technique, difficulty, ABV, pro tips)
- Polished dark theme interface
- Enhanced information hierarchy

## Tech Stack

- **Frontend:** React 19
- **Styling:** Tailwind CSS
- **State Management:** React Hooks (useState, useEffect)
- **API:** TheCocktailDB
- **Design System:** shadcn/ui patterns (v4)

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/mixmaster.git
cd mixmaster

# For v3
cd v3
npm install
npm start

# For v4
cd v4
npm install
npm start
```
